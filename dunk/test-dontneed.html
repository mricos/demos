VMX8 Modes

CC Mode Shift + Right

  • Faders send CC messages (7-bit, 0-127)
  • Example: Fader 1 = CC 47, Fader 2 = CC 43
  • Buttons send CC with press/release
  • Good for: generic MIDI mapping, synths, effects

DAW Mode Shift + Left

  • Faders send Pitch Bend (14-bit, 0-16383)
  • Ch 1-8 pitch bend = Faders 1-8
  • Buttons send Note On/Off (MCU protocol)
  • Good for: DAWs, LED feedback, high-res faders
LED control only works in DAW Mode with MCU Session active. The VMX8 has limited LED feedback - only Mute Ch1 (0x10) and Play (0x5E) respond.
