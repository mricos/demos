VMX8 Modes

CC Mode Shift + Right

  • Faders send CC messages (7-bit, 0-127)
  • Fader CCs: 47, 43, 39, 35, 31, 27, 23, 19
  • Knob CCs: 46, 42, 38, 34, 30, 26, 22, 18
  • Buttons send CC 127 on press, 0 on release
  • Good for: generic MIDI mapping, synths, effects

DAW Mode Shift + Left

  • Faders send Pitch Bend (14-bit, 0-16383)
  • Ch 1-8 pitch bend = Faders 1-8
  • Knobs send relative CC (63=left, 65=right)
  • Buttons send Note On/Off (MCU protocol)
  • Good for: DAWs, LED feedback, high-res faders

LED Feedback

Working LEDs: Mute Ch1 (note 16/0x10), Play (note 94/0x5E)

Not working: Rec (0x00-07), Solo (0x08-0F), Mute 2-8 (0x11-17), Select (0x18-1F)

Protocol: Send Note On velocity 127 = LED on, velocity 0 = LED off

Anomaly: Note 115 (0x73) also controls Mute Ch1 LED

VMX8 Hardware
  • 8 motorized touch-sensitive faders
  • 8 endless rotary encoders (knobs)
  • 32 backlit buttons (4 rows × 8 channels)
  • Transport section with 5 buttons
  • USB-C + Bluetooth connectivity
  • Ports: SINCO VMX8-Private (control), SINCO VMX8-Master (general)
